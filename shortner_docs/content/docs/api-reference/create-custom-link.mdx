---
title: Create custom link
api: POST /link/create-custom-link
---

Create a shortened WarpIt link with a custom back-half code.

### Body

- `originalUrl` (`string`, required): The long URL you want to warp.  
  Example: `https://google.com`
- `customUrlCode` (`string`, required): The custom code you want to use for the short URL.  
  Example: `my-custom-link`

### Response

- `success` (`boolean`): Indicates if the request was successful.
- `data` (`object`)
  - `shortUrl` (`string`): The generated WarpIt link with your custom code.
- `message` (`string`): A descriptive success message.

#### Success example

```json
{
  "success": true,
  "data": {
    "shortUrl": "http://localhost:4000/my-custom-link"
  },
  "message": "Here is your Short Url",
  "status": 200
}
```
#### Error example

```json
{
  "success": false,
  "message": "Unfortunately, This custom code is already taken",
  "status": 409
}
```
